3 NEURON is a simulator for models of neurons and networks of neuron.
4 See http://neuron.yale.edu for installers, source code,
5 documentation, tutorials, announcements of courses and conferences,
6 and a discusson forum.

8 ## Building NEURON
9 
10 
11 ### Building using autotools
12 
13 Basic installation on Linux from the nrn...tar.gz file is:
14 
15  Build InterViews first from the git repository at
16  http://github.com/neuronsimulator/iv or the iv...tar.gz file at
17  http://neuron.yale.edu/ftp/neuron/versions/alpha/
18 
19 ```
20  ./configure
21  make
22  make install
23 ```
24 
25 If sources are obtained from the git repository,
26 http://github.com/neuronsimulator/nrn ,
27 create the automake, autoconf, libtool generated files by:
28  sh build.sh
29 
30 Particularly useful configure options:
31 
32 
33 - `--prefix=/some/path` : Install in in this location of your filesystem
34 - `--without-x` : If the InterViews graphics library is not installed.
35 - `--with-iv=/where/you/installed/it` : If iv was not installed in <prefix>/../iv
36 - `--with-paranrn` : Parallel models on cluster computers using MPI (openmpi or mpich2)
37 - `--with-nrnpython` : Use Python as an alternative interpreter (as well as the native HOC interpreter).
38  This is required to use the Reaction-Diffusion extension.
39 
40 
41 For more details see the [INSTALL.md](https://github.com/neuronsimulator/nrn/blob/master/INSTALL.md)
42 file.

44 ### Building using cmake
45 
46 NEURON can now also be built and installed using cmake. Currently we are evaluating the two build
47 system options but given positive feedback from the community we plan to depricate autotools and
48 completely switch to cmake. Therefor we would be grateful for any feedback.
49 
50 To build with cmake we require following packages to be installed on the system:
51 
52 - bison
53 - cmake >=3.3.0
54 - flex
55 - A C/C++ compiler suite (GCC, Intel Parallel Studio, clang,
56 
57 Following packages are optional:
58 
59 - python >=2.7, or python >=3.5
60 - cython
61 - MPI
62 - X11 (or XQuartz on macos)
63 
64 
65 1. Download the NEURON `.tar.gz` or clone from https://github.com/neuronsimulator/nrn
66 
67  ```
68  git clone https://github.com/neuronsimulator/nrn
69  cd nrn
70  ```
71 
72 2. Create a build directory
73 
74  ```
75  mkdir build
76  ```
77 
78 3. Run cmake with the appropriate options
79 
80  ```
81  cmake -DNRN_ENABLE_CORENEURON=OFF -DNRN_ENABLE_PYTHON=ON -DNRN_ENABLE_MPI=OFF -DNRN_ENABLE_INTERVIEWS=ON ..
82  ```
83 
84 4. Build the code
85 
86  ```
87  make -j
88  ```
89 
90 For more installation information see:
91 http://neuron.yale.edu/neuron/download/getdevel
